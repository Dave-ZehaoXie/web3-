## 🔍 深度解读：Bitcoin Whitepaper

<details>
<summary><b>📌 点击展开：比特币到底是什么？（白皮书精华解读）</b></summary>

### 一、 核心痛点：传统电子支付的麻烦
平时网上转账都得靠银行、支付宝这些第三方机构，存在以下问题：
* **中介成本**：必须信任第三方，产生手续费，小额转账不划算。
* **不可逆性差**：交易可能被撤回，商家需防范诈骗。
* **双花问题 (Double-Spending)**：没有第三方时，难以确认同一笔钱是否被重复支付。

> **核心目标**：不用任何信任的第三方，实现点对点转账，确保交易不可篡改且不可逆。

### 二、 关键设计：如何实现“无信任转账”？

#### 1. 电子货币的本质：数字签名链
比特币是一串数字签名形成的 `ownership` 链：
* **流转**：A 用私钥签名（上一笔记录 + B 的公钥）转给 B。
* **验证**：任何人都能通过公钥验证签名，确认归属合法。

#### 2. 解决“双花”：PoW 与 最长链机制
* **公开账本**：所有交易全网广播。
* **工作量证明 (PoW)**：节点竞争计算复杂哈希题，只有胜者能打包区块。
* **共识规则**：全网只认 **“最长的区块链”**。篡改历史需要超过全网 51% 的算力，成本极高，保证了安全性。

#### 3. 激励机制：挖矿与手续费
* **区块奖励**：打包区块的节点获得新发行的比特币（目前每 4 年减半）。
* **交易费**：系统总量恒定（2100万），后期节点主要靠手续费维持。

#### 4. 隐私与安全
* **匿名性**：使用公钥地址而非真实身份，建议每笔交易更换新地址。
* **确认时间**：通常建议等待 `6个区块` 确认，此时篡改概率几乎为 0。

### 三、 总结
比特币是一套 **“全网共同维护、用算力担保、不可篡改的公开账本”**。它用“数学和算力”替代了对“人或机构”的信任。
