## 🔍 论文深度解读

### 1. 🪙 Bitcoin Whitepaper 解读
<details>
<summary><b>点击展开：比特币到底是什么？</b></summary>

* **核心痛点**：解决传统支付依赖第三方中介、存在双花风险的问题。
* **设计逻辑**：通过“数字签名链”确定归属，通过“PoW（工作量证明）”竞争记账。
* **安全保障**：最长链即正版账本，篡改成本极高。
* **总结**：它是全网共同维护、不可篡改的公开账本，用算力替代了信任。
</details>

### 2. 💎 Ethereum Whitepaper 解读
<details>
<summary><b>点击展开：以太坊——区块链的“操作系统”</b></summary>

#### A. 比特币的“局限”
比特币像是一个功能单一的“计算器”，只能处理转账。它无法实现复杂的逻辑（如自动分红、限额支付），因为其脚本语言功能有限，且无法记录复杂的“状态”。

#### B. 以太坊的核心升级
以太坊不仅管“钱”，还管“逻辑”与“状态”：
* **图灵完备语言**：内置编程语言，能实现任何复杂的业务逻辑（循环、判断）。
* **状态机模型**：不仅记录余额，还记录合约的中间状态（如投票数、到期时间）。

#### C. 关键概念
* **账户体系**：分为外部账户（人控制）和合约账户（代码控制）。
* **智能合约**：无人看管的自动机器，代码即法律。
* **Gas（燃料）**：为了防止死循环和恶意攻击，每步操作都需消耗 Gas。

#### D. 应用场景
* **代币发行**：几行代码即可发行自己的资产或积分。
* **DAO（去中心化组织）**：靠代码定规则，自动执行集体决策。
* **去中心化存储与金融**：打破中心化平台的垄断，实现自动赔付、对冲风险等。

> **核心总结**：比特币是去中心化货币，以太坊是去中心化应用平台。它让区块链从“只能转账”进化到“能跑万物”。
</details>
